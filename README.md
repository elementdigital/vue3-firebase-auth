# vue3-firebase-auth
Vue Firebase authentication system

## Project setup
```
npm install
```
* add your firebase credentials to env.example
* save as env.local

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
